Comando PS

Muestra los procesos que se están ejecutando en el sistema.

Opciones Principales

  • -e: Muestra todos los procesos.
  • -f: Muestra formato completo (UID, PID, etc.).
  • -u: Muestra procesos de un usuario específico.
  • aux: Muestra todos los procesos con máximo detalle.

Ejemplos de uso

  • Ver todos los procesos: ps aux
  • Ver procesos del usuario root: ps -u root
  • Ver árbol de procesos: ps -ef
